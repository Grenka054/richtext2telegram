How to use:

  • Select your Telegram version (Desktop/Web) using the switcher in header
  • Copy formatted text from any application (Word, Google Docs, web pages)
  • Paste it into the left panel - you should see the formatting preserved
  • Copy the Telegram-formatted text from the right panel
Important: Telegram Desktop and Web versions handle combined formatting (bold+italic) differently. Use the switcher to match your client version.

Desktop Version Formatting

  • Bold: **text**
  • Italic: __text__
  • Monospace: ```text```
  • Strikethrough: ~~text~~
  • Spoiler: ||text||

Web Version Formatting

  • Bold: **text**
  • Italic: __text__
  • Bold + Italic: **__text__**
  • Monospace: ```text```
  • Strikethrough: ~~text~~
  • Spoiler: ||text||
